Plugin for putting Ads in H tags and P tags inside articles and top of article and bottom of article. 
Also supports AMP.

== Description ==

This plugin insert ads code such as adsense inside articles.

Search H tags at the middle of the article and insert the ads code.
If can not insert with H tag,going to search and Insert with P tag.

Ads code sort by the access with desktop, mobile, AMP.

From Version 1.1 added these function
Ads in top of Article
Ads in bottom of Article
Ads insert tag. 

More infomation see [WordPress plugin Ads Into Post](https://b.eax.jp/wp/wp-plugin/15939/#en)



== Installation ==

1. Upload the "ads-into-post" folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
